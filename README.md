# Profile Page Using CSS Box Model

## Task Description
This project demonstrates styling a profile page using the CSS Box Model concepts
such as margin, padding, border, and content.

## Technologies Used
- HTML
- CSS
- VS Code
- Browser Developer Tools

## Features
- External CSS file
- Class-based styling
- Proper spacing using box model
- Clean and readable layout

## Learning Outcome
- Clear understanding of CSS box model
- Difference between margin and padding
- Avoiding inline CSS
- Using reusable CSS classes
